HTML Links | Hyperlink

Tautan HTML adalah hyperlink. Anda dapat mengklik tautan dan melompat ke dokumen lain. Saat anda menggerakkan mouse ke atas tautan, panah mouse akan berubah menjadi tangan kecil.

Note : Tautan tidak harus berupa teks. Tautan dapat berupa gambar atau elemen HTML lainnya!
